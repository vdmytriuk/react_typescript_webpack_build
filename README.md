# React - TS - Webpack build

### How to start development:
`npm i`

`npm run dev`

### How to start build:
`npm run build:prod`

## Technology stack
- **UI**: `React`, `SCSS`
- **Lang**: `Typescript (4.9+)`
- **Linters**: `stylelint`, `eslint`
